## Installation

```bash
$ pnpm install
```

## Third Party Dependencies

```bash

# prisma
# Development
$ pnpx prisma migrate dev || npx prisma migrate dev

$ pnpm run seed
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
